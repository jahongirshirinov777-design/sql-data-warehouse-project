# SQL Data Warehouse: End-to-End Sales Analytics

## 📊 Project Overview
This project demonstrates the architecture and implementation of a professional modern **Data Warehouse**. It showcases the transformation of raw transactional data into a high-performance **Star Schema** optimized for business intelligence.

### Core Competencies:
* **ETL Pipeline:** Automated extraction, cleaning, and loading of raw datasets.
* **Dimensional Modeling:** Implementation of Fact and Dimension tables for efficient querying.
* **Process Automation:** Using **Stored Procedures** and **Views** to streamline reporting logic.
* **Data Integrity:** Handling complex NULL logic and enforcing rigorous data types.

## 🛠️ Tech Stack
* **Engine:** SQL Server (T-SQL)
* **Advanced Logic:** CTEs, Window Functions, Schema Design, and Procedure Automation.
